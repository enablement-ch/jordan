# Output Format

The scorecard has three layers:

1. **At-a-glance table** - so the rep can screenshot it and act on it.
2. **Per-dimension coaching** - one block per dimension with score, why, what to improve, and the exact questions to ask next call.
3. **Process discipline check** - did they book the next call? This is a non-negotiable.
4. **The biggest gap + next move** - the single thing to fix.

## Template

````markdown
# Jordan's Scorecard - {Prospect Company} / {Call Date}

**TL;DR:** {One sentence verdict. Be honest. Example: "Solid discovery, dead on critical event - this deal is in Q4 territory unless you fix it."}

---

| Dimension | Score | Headline |

| **Critical Event** | X/10 | {one-line summary} |
| **Decision** | X/10 | {one-line summary} |

**Overall: X/10**

**Next call booked?** ✓ Yes / ✗ No - {one-line verdict}

## Process Discipline Check

**Was the next call booked on the calendar before this one ended?**

{If yes:} ✓ **Locked.** {One sentence on what was booked and when.}

{If no:} ✗ **Not booked.** This is the single biggest red flag in this call, regardless of the SPICED scores. Without a confirmed next slot, every gap below becomes irrelevant - you can't fix Critical Event in a meeting that doesn't exist.

**Fix:** Before you hang up next time, end with: "Quick logistics - I want to make sure we keep this moving. Can we lock {specific day} at {specific time} for the next conversation?" Then send the calendar invite while still on the call.

---

## The Biggest Gap

**{Dimension} - X/10.** {OR if no next call was booked: "You hung up without a calendar invite. Fix that before anything else."}

{2-4 sentences. Name the dimension dragging the deal. Reference the moment it broke. Don't restate the score - explain the consequence.}

---

## Your Next Move

{The single most important thing to do before or during the next interaction. If a next call isn't booked, the next move is to book it. If a next call IS booked, the next move is the discovery question that closes the biggest gap.}

{If the move is verbal:}
**Say this:**
> "{Exact words.}"

{If the move is an action (book a stakeholder, prep an asset, etc.):}
**Do this:** {specific action}

---

## Bonus Notes (optional)

{Only include if there's something outside SPICED + process discipline worth flagging. Examples: a competitor mention, a buying signal you talked over, a tone shift, a champion forming. Skip this section entirely if there's nothing to add.}
````

## Rendering rules

- **The at-a-glance table is the screenshot.** Keep headlines tight - one line, no jargon.
- **Per-dimension blocks are the coaching.** This is where the work gets done. Three to five questions per dimension, max - more becomes overwhelming.
- **Quotes must be real.** If a dimension wasn't covered, write "(not addressed)" - don't fabricate.
- **The next-call check is mandatory.** Always. If skipped, the rep doesn't see why their deals stall.
- **One biggest gap, not three.** A rep can fix one thing per call. Three is paralysis.
- **No emoji, no exclamation points.** Clinical layout, brutal content.
- **Skip the bonus section** if it would be filler.

## When the call doesn't have enough material

If the transcript is too short, cut off, or clearly mid-conversation, output the scorecard with whatever is scorable and add a single line at the top:

> **Note:** This appears to be a partial transcript. Scoring is based on what's available - some dimensions may be unfairly low.

Don't refuse to score. Just flag it and proceed.
